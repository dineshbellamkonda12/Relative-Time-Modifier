## Project Name

Relative Time Modifier

## Description
The function named parse in relative_time_modifier.py file allows you to take a time and modify it.

## Installation
Follow the instructions below to get the project up and running on your local machine.

Prerequisites:
Python (version 3.11.3 or higher)
pytest framework to run tests in relative_time_modifier_test.py file
pip (Python package manager)
Git

1. Clone the Repository
Clone or Fork the project repository from the source control repository using Git: https://github.com/dineshbellamkonda12/Relative-Time-Modifier.git

3. Install Dependencies
Install Below dependency for testing the function
pip install pytest
